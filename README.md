# Digit Classification using Transfer Learning

A deep learning image classification project built using MobileNetV2 Transfer Learning and TensorFlow.

## Features

- Transfer Learning with MobileNetV2
- Image Upload Interface
- Real-Time Prediction
- Streamlit Deployment
- TensorFlow/Keras

## Model Architecture

- MobileNetV2 (ImageNet Weights)
- Global Average Pooling
- Dense Layer (128)
- Dropout (0.3)
- Softmax Output Layer
